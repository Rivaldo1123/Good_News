GitHub Auto-Publish
Save changes directly to GitHub — Netlify deploys automatically

Setup (one time):
1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Click Generate new token
3. Set repository access to Rivaldo1123/Good_News
4. Under Permissions → Repository → Contents: Read and Write
5. Generate & paste the token below

Note: Your token is stored only in this browser — never sent anywhere except GitHub's API.
