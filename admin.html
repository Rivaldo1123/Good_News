Security & GitHub
Authentication and publish settings

Authentication is handled by Netlify Identity. Manage users in your Netlify dashboard under Identity.

Roles: In Netlify → Identity → select a user → Edit metadata → add {"roles":["admin"]} to grant admin access.

GitHub Auto-Publish

Setup: Create a fine-grained PAT at GitHub → Settings → Developer settings with Contents: Read/Write on Rivaldo1123/Good_News, then add GITHUB_TOKEN in Netlify → Environment variables.

Scheduled Publishing
Auto-publish the draft at a future time

The scheduler checks every 15 minutes. When the time passes, the Draft config publishes automatically via Netlify cron. Requires GITHUB_TOKEN.
